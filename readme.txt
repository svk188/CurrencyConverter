При запуске приложения всплывает окно программы, в котором есть 4 поля для ввода:
	1) "Из валюты" - предполагает ввод буквенного кода исходной валюты.	
	2) "У меня есть" - предполагает ввод суммы исходной валюты.
	3) "В валюту" - предполагает ввод буквенного кода желаемой валюты.
	4) "Я получу" - вывод результата конвертирования.

Для взаимодействия с окном программы в правом верхнем углу находятся 3 кнопки:
	1) "Свернуть" - кнопка для сворачивания окна в панель задач.
	2) "Развернуть" - кнопка для перехода программы в полноэкранный режим и обратно в оконный.
	3) "Закрыть" - кнопка для завершения работы с приложением.

Для расчета курса валют используются следующие буквенные коды валют: 
	USD,JPY,BGN,CYP,CZK,DKK,EEK,GBP,HUF,LTL,LVL,MTL,PLN,ROL,RON,SEK,SIT,SKK,CHF,ISK,NOK,HRK,RUB,TRL,TRY,AUD,BRL,CAD,CNY,HKD,IDR,ILS,INR,KRW,MXN,MYR,NZD,PHP,SGD,THB,ZAR
С расшифровкой данных буквенных кодов можно ознакомиться на сайте http://vch.ru/zifrovye_i_bukvennye_kody_valyut_mira.html.
